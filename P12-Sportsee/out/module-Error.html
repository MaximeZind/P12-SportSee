Module: Error

Composant Error - Représente la page d'erreur 404 de l'application SportSee. Ce composant affiche la page d'erreur 404 lorsque l'utilisateur accède à une URL inexistante. Il affiche un message d'erreur et un lien pour retourner à la page d'accueil. Le titre de la page est mis à jour avec "SportSee - Error 404".
